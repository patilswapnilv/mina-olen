Mina olen WordPress theme

Mina olen is a business type of theme. It includes support for Easy Digital Downloads, Custom Content Portfolio, Testimonials by WooThemes and Whistles support.

This theme is a parent theme. All modification should be made via a child theme.

Features

Mina olen is built on Hybrid Core framework. There is a brilliant minded man called Justin Tadlock who has created it. Remember also to check out all the Foxnet Themes.

  • Per-post layouts which means that you can choose one or two column layout. Content on the left or right. Default is 2 column layout, content on the left.
  • Post setting that allows you to add a custom stylesheet for individual posts.
  • A smart breadcrumb system.
  • Built-in pagination.
  • Post thumbnails that work with WordPress featured image functionality and provide extra features.
  • Multiple and flexible sidebars areas: Header, Primary and Subsidiary template widgets.
  • Dynamic body, post, and comment CSS classes.
  • Fully internationalized, so you can translate the theme in any language. You should place your translations files to child theme languages folder.
  • Built-in WordPress menu system. There are three menus in the theme: Primary, Subsidiary and Social. Note! Primary menu only support for one level sub menu items, subsidiary only top level menu items.
  • Global layout can be changed under Appearance »» Customize »» Layout.
  • Set logo under Appearance »» Customize »» Logo Upload.
  • Set Callout text and link in Header extra info metabox when you edit your page.
  • Set background under Appearance »» Background.
  • Set header image under Appearance »» Header.
  • Ability to create custom templates for any post type in multiple ways.
  • Highly-configurable widgets that give you full control over their display.
  • Widget areas are content-aware, meaning that they only appear if they contain active widgets.
  • Support for all Post Formats.
  • Theme was designed with Mobile First attitude.

Child themes

Mina olen is a parent theme, so use a child theme for any customizations. You can also download blank child theme in theme page. If you want do it yourself, here is how to get started.

  • Create a theme folder in your /wp-content/themes directory called mina-olen-child.
  • After that create a style.css file in /wp-content/themes/mina-olen-child folder.
  • At the top of your style.css file, add the below information.
/**
 * Theme Name: Mina olen Child
 * Theme URI: http://link-to-your-site.com
 * Description: Describe your child theme.
 * Version: 0.1
 * Author: Your Name
 * Author URI: http://link-to-your-site.com
 * Tags: Some, Tags
 * Template: mina-olen
 */

There is no reason to import parent theme styles, because they are loaded automatically. Note! If you compress your style.css and copy your styles to a file called style.min.css, it will be used automatically.

Using child theme is officially-supported way of modifying themes in WordPress.

Supported plugins

Custom Content Portfolio

If you need to show off your portfolios, install Custom Content Portfolio Plugin and you're good to go.

Your portfolios will appear in URL domain.com/portfolio. Remember to flush rewrite rules first by going Settings »» Permalinks.

Easy Digital Download

If you need to sell your digital downloads Easy Digital Download is the best plugin for that.

Your downloads will appear in URL domain.com/downloads.

Testimonials by Woothemes

If you need to show testimonials on yor site, install and activate Testimonial by Woothemes. Your testimonials will appear in URL domain.com/testimonials.

Gravity Forms

Mina olen includes base CSS for Gravity Forms plugin.

Theme Settings (Appearance »» Customize)

All of the theme setting is under Appearance »» Customize.

Set logo

Set logo under Appearance »» Customize »» Logo Upload.

Change default layout

Mina olen theme has two column layout by default. You can change this default layout under Appearance »» Customize »» Layout. You can overwrite this in post editor by choosing layout for that certain post:

  • 1 Column
  • 2 Columns: content / sidebar (theme default)
  • 2 Columns: sidebar / content

Footer text

You can change Footer text in Footer text section.

Header Image

By default there is a static image in the header. You can overwrite default header image in singular page by setting right size featured image. Feature header size is 1920 x 379 pixels.

Page Templates

There is one page templates in this theme called Front Page. It's located in pages folder. If you need to overwrite page template in a child theme, create subfolder pages in child theme and copy page template in that folder and start modifying.

You can select which pages you want to show in Front page and do you show posts, downloads or testimonials.

You can select a page template from the "Template" section of the "Attributes" meta box when writing or editing a page.

If you want to use Front Page template as a front page for your site instead of the standard blog post view, follow these instructions.

Creating the front page

  • First create a new page. You can name it anything. For example "Home" or "Portfolio".
  • When creating this page, select "Front Page" from the "Template" drop-down inside of the "Attributes" meta box.
  • Click the "Publish" button.

Change site front page

  • Go to the "Settings »» Reading" screen in the admin to adjust the page shown on the front of the site.
  • Locate the section labeled "Front page displays".
  • Select "A static page".
  • In the "Front Page" drop-down, select the page you created earlier that has the "Front Page" template.
  • Click the "Save Changes" button.

Flexible subsidiary and front page widgets

You can use 1, 2, 3, 4 or 5 widgets in subsidiary and header widgets areas and the layout is still gonna look nice. The original idea having flexible and responsive widget areas is from Sukelius Theme.
